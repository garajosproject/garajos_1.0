# GarajOS

GarajOS is a multi-surface mobility platform for riders, garages, sellers, and admins.

## Repository Structure

```text
garajos/
├── apps/
│   ├── rider-app/
│   ├── garage-app/
│   └── admin-panel/
├── backend/
│   ├── api/
│   ├── services/
│   └── prisma/
├── packages/
│   ├── ui/
│   ├── config/
│   └── utils/
├── docs/
├── .env
├── package.json
└── tsconfig.base.json
```

## Recommended Stack

- Rider app: React Native / Expo
- Garage app: React Native / Expo
- Admin panel: Next.js + Material UI
- API: NestJS
- Database: PostgreSQL + Prisma
